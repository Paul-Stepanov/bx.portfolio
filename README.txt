ЗАДАНИЯ:

1️⃣ Развернуть битрикс с дефолтным шаблоном. Можно у себя локально, можно на каком-нибудь сервере.

2️⃣ Настроить git, вынести в исключения папки bitrix, upload, images, файлы htaccess, robots и прочие файлы, которые не
должны попадать в репозиторий. Создать ветку для внесения изменений, руководствоваться gitflow.

3️⃣ Написать самописный компонент Отзывы и вывести его на странице. Для хранения отзывов использовать инфоблок.
В компоненте необходимо реализовать вывод списка отзывов в произвольном формате (имя, текст отзыва, оценка) и форму для отправки отзыва.

4️⃣ Написать событие на изменение элемента инфоблока Отзывы. При изменении/пересохранении элемента Поле "Текст отзыва" и
 "Оценка" не должно меняться. Можно пользоваться стандартными полями элемента инфоблока, можно создать свои поля.

5️⃣ Закоммитить изменения, залить репозиторий на github.
